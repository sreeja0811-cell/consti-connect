# Consti-Connect: AI-Powered News & Constitution Linker

## 📌 Overview
A full-stack application that connects constitutional articles with real-time news using NLP techniques.

## 🚀 Features
- Real-time news integration via API
- NLP-based keyword extraction
- Mapping news to constitutional provisions
- Interactive user interface

## 🛠 Tech Stack
- React.js
- Node.js, Express
- SQL
- NLP

## 📁 Project Structure
- Frontend (React)
- Backend (Node.js)
- NLP Service

## ▶️ How to Run
1. Install dependencies
2. Start backend server
3. Run frontend
